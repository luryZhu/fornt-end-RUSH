  - 实现一个repeat函数，repeat(func,times,wait)。具有以下功能const repeatFunc = repeat(alert, 4,300),repeatFunc('hello world'),alert四次‘hello world'每次间隔300ms

    ```js
    function repeat(func, times, wait){
        return ()=>{
            for(let i=0;i<times;i++){
                setTimeout(()=>{
                           func(...arguments)
                           },wait*i)
            }
        }
    }
    ```
